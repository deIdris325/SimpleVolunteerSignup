# OpenSignup Board
A simple self-hosted signup board for tasks, events and volunteer slots with reserve lists.

## Default credentials

- Admin password: `initpasswort`
- Access code: `initpasswort`

Change both values in `config.php` before publishing publicly.

## Features

- JSON-based schedule storage
- Public signup view
- Reserve list support
- Admin area for managing entries
- Editable public texts in the admin area
